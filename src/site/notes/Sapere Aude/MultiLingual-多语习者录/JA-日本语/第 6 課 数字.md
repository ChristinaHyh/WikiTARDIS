## 1-99

| 数字  | 读法1     | 读法2          |
| --- | ------- | ------------ |
| 0   | れい      | ゼロ           |
| 1   | いち      |              |
| 2   | に       |              |
| 3   | さん      |              |
| 4   | し       | よん           |
| 5   | ご       |              |
| 6   | ろく      |              |
| 7   | しち      | なな           |
| 8   | はち      |              |
| 9   | きゅう     |              |
| 10  | じゅう     |              |
| 11  | じゅういち   |              |
| 20  | にじゅう    |              |
| 47  | よんじゅうしち | よんじゅうなな      |
| 74  | ななじゅうし  | ななじゅうよん（更常用） |
- 关于0，数数一般用ゼロ，房间号可以用ゼロ也可以用まる
- 十位数的4和7分别只能用よん和なな
